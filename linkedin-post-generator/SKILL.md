# LinkedIn Post Generator Skill

Complete LinkedIn strategy with 49+ formulas, 4 templates, and 15 practical tips.

## Features

✅ 49+ Formulas by category and objective
✅ 4 Ready-made templates
✅ 15 Practical tips for viral posts
✅ Pre-publishing validation
✅ Auto-optimization
✅ Engagement scoring

## Install
```
npm install
```

## Usage
```javascript
const LinkedInPostGenerator = require('./index.js');
const generator = new LinkedInPostGenerator();

const post = await generator.generatePost({
  objective: 'visibility',
  formulaType: 'mantra-repetible',
  topic: 'Your idea',
  details: ['detail1', 'detail2']
});
```
